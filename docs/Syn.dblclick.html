dblclick() Syn.dblclick

 

Syn.dblclick(options, element, callback)

Dblclicks an element. This runs two click events followed by a dblclick on the element.

Example

Syn.dblclick({},'open')

Parameters

  1. options {Object}

  2. element {HTMLElement}

  3. callback {function()}
